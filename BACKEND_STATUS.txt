1. Send Message
   POST /api/chat/
   Body: { message, agent, session_id, is_paid }
   Response: { success, session_id, response, message_count }

2. Get Session
   GET /api/session/?session_id=xxx
   Response: { session_id, agent, message_count, messages }

3. Unlock Access
   POST /api/unlock/
   Body: { session_id, password }
   Response: { success, message }

🛠️ TROUBLESHOOTING QUICK LINKS

Issue: Command not found
  → Make sure VENV is activated: source venv/bin/activate

Issue: Port 8000 already in use
  → Use different port: python manage.py runserver 8001

Issue: "Agent not found"
  → Initialize agents: python setup_db.py

Issue: Password unlock doesn't work
  → Check .env file: cat .env | grep CHAT_PASSWORD

Issue: API returning 500 error
  → Check server logs in terminal

For more help:
  → See API_DOCUMENTATION.md
  → Check Django docs: https://docs.djangoproject.com/
